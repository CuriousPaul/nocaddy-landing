# PRD v1.0 — 골프 자동 예약 컨시어지
> 작성일: 2026-04-20 | 작성자: 파이토스 + 진배

---

## 1. 문제 정의

### 페인포인트
한국에서 2인 노캐디 골프를 치고 싶은 사람들은 구장을 "찾기"는 어렵지 않다.
하지만 **원하는 날짜에 예약하는 것**이 극도로 귀찮고 어렵다.

**구체적 페인:**
- 구장마다 예약 오픈 날짜/시간이 다름 (화요일 17:30, 4주 전 월요일 AM9시 등)
- 오픈 즉시 수분 내 마감 → 타이밍 놓치면 끝
- 잔여티 노리려면 매일 새벽 직접 확인해야 함
- 전화 예약만 되는 구장 다수 → 업무 중 전화 어려움
- 성수기(10~11월) 수도권 2인은 사실상 예약 불가

**결과:** 많은 골퍼들이 일본으로 날아간다 (항공+숙박+골프 = 50~100만원)

---

## 2. 솔루션

### 핵심 가치 제안
> "원하는 구장 말해놓고 Deposit 걸어두면, 예약 오픈 즉시 자동으로 잡아드립니다."

**성공 시:** 수수료 청구 (3~6만원)
**실패 시:** Deposit 환불 or 다음 시도에 이월

---

## 3. 타겟 유저

### 페르소나
- **골프 동호인** (30~50대, 자산 있음)
- 2인(부부/커플/친구 1명)으로 라운딩을 원함
- 노캐디 선호 (비용 절감 + 자유로움)
- 골프에 돈 쓰는 걸 아까워하지 않음 (6만원 컨시어지 비용 기꺼이 지불)
- 시간이 아까운 사람 (예약에 쏟는 시간 절약 원함)

### 수요 규모
- 일본 골프 관광 수요 = "한국에 노캐디 2인 구장 부족"의 직접적 증거
- 수도권 주요 2인 구장: 예약 오픈 즉시 마감 → 대기 수요 실재

---

## 4. 기능 정의

### MVP (Week 1)
| 기능 | 설명 | 구현 방식 |
|------|------|-----------|
| 구장 + 날짜 요청 등록 | 유저가 원하는 구장/날짜/조건 입력 | 카카오톡 채널 or 웹폼 |
| Deposit 결제 | 예약 시도 보증금 선결제 | 카카오페이 or 토스 |
| 자동 예약 시도 | 오픈 시각에 봇이 예약 | 웹 스크래핑 + 자동화 |
| 결과 알림 | 성공/실패 여부 즉시 알림 | 카카오톡 메시지 |
| 정산 | 성공 시 수수료, 실패 시 환불/이월 | 자동 정산 |

### Phase 2 (PMF 이후)
- AI 콜: 전화 예약만 되는 구장 자동 전화
- 예약 오픈 모니터링 확대 (전국 100개)
- 숙박 패키지 연동
- 구장별 예약 성공률 공개

## 5. 경쟁사 분석

| 서비스 | 유형 | 차이점 |
|--------|------|--------|
| 카카오골프예약 | 예약 플랫폼 | 입점 구장만, 2인 노캐디 필터 없음, 자동화 없음 |
| 김캐디 | 정보 제공 | 예약 대행 없음, 정보 제공만 |
| 일본 Rakuten Gora | 예약 플랫폼 | 일본 전용, 한국 없음 |
| 공연 자동 예약 봇 | 티켓팅 자동화 | 골프 특화 없음, 약관 위반 가능성 |
| **우리** | **자동 예약 컨시어지** | **골프 특화 + Deposit 모델 + 성공 보수** |

### 핵심 차별점
기존 플랫폼: "구장을 보여준다"
우리: "예약을 대신 잡아준다" → 결과로 돈 받음

## 6. 수익 모델

| 항목 | 금액 | 근거 |
|------|------|------|
| 예약 성공 수수료 | 3~6만원/건 | 유저 직접 말함 ("6만원도 낼 수 있음") |
| Deposit | 1~2만원 | 진지한 수요 필터링 |
| 월 구독 (Phase 2) | 3~5만원/월 | 반복 유저 대상 |

**Week 1 목표:**
- 5건 예약 성공 = 5 × 5만원 = 25만원 수익
- 이게 PMF 증명
